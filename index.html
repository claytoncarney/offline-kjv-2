Design

This is an Offline-First, Mobile-First application. Once downloaded, it provides full functionality, even when no internet connection is available.

It supports any Android, Windows, or Linux platform with the latest version of Google Chrome or Mozilla Firefox web browser.

On an Android platform, the best user experience is available using Chrome for Android. By selecting 'Add to Home screen', the application mimics a native Android app.

Navigation And Glyphs

The application uses five Panes: Read, Contents, Bookmarks, Search, and Help.

Each Pane has one or more Pages.

Use the Back button to return to the previous Pane/Page:

Back

On smaller devices, only one Pane is visible at a time. Tap the desired Pane button to display that Pane. Tap the Back button to return to the Read Pane.

On larger devices, the Read Pane is always visible and a single sidebar displays either the Contents, Bookmarks, Search, or Help Pane. Tap the desired Pane button to display that Pane in the sidebar.

The Read Pane has a Navigation Bar at the bottom.

The Contents, Bookmarks, Search, and Help Panes have a Toolbar at the top and a Navigation Bar at the bottom. These provide context information and buttons to access other features.

Read Pane

The Read Pane has one Page: Read.

Read Page

The Read Page presents a list of verses from the active Chapter of the active Tome.

The active Tome is set via the Tomes Page.

The active Chapter is set via the Contents Page, Bookmarks Page, or Results Page.

For more details, see the Contents, Bookmarks, and Search Help Topics.

Tap the Single, Double, or Triple Column button (if present) in the Navigation Bar to set the number of columns.

Bookmarks

Bookmarks in the active Folder for the active Chapter are highlighted on the Read Page.

Tap on a verse text to add/remove a Bookmark in the active Folder.

Clipboard

Tap a verse number to copy that verse to the system clipboard.

Contents Pane

The Contents Pane has two Pages: Contents (default) and Tomes.

From the Contents Pane you select the active Tome and active Chapter.

Contents Page

The Contents Page Toolbar displays the name of the active Tome.

The Contents Page presents the Book list for the active Tome.

Tap a Book name to display the Chapter list of that Book. Tap a Chapter number to set the active Chapter. The Read Page is opened at the active Chapter.

Tomes Page

Tap the Tomes button in the Contents Page Toolbar to open the Tomes Page.

The Tomes Page presents the Tome list.

Tap a Tome name to set the active Tome. The Contents Page is opened with the Book list of the active Tome.

Tap the Back button to return to the Contents Page.

Bookmarks Pane

The Bookmarks Pane has six Pages: Bookmarks (default), Add Folder, Folders, Move/Copy, Rename Folder, and Delete Folder.

Bookmarks Folders

A set of Folders is maintained for each Tome. This facilitates grouping of Bookmarks for various purposes.

The active Folder set is selected when the active Tome is set.

Bookmarks Page

The Bookmarks Page Toolbar displays the name of the active Folder.

The Bookmarks Page presents the Bookmarks list for the active Folder.

Tap a Bookmark to open the Read Page at the selected verse.

Bookmarks are managed via their Action Menu. Tap the Menu button to open the Action Menu. You may reorder, move/copy, or delete Bookmarks as desired.

Tap the Sort Ascending or Sort Descending button in the Navigation Bar to sort Bookmarks in the active Folder.

Add Folder Page

Tap the Add Folder button in the Bookmarks Page Toolbar to open the Add Folder Page.

Enter a Folder name and tap the Save button to add a new Folder to the active Tome Folder set. The active Folder is set to the new Folder and the Bookmarks Page is opened.

To cancel, tap the Back button to return to the Bookmarks Page.

Folders Page

Tap the Folders button in the Bookmarks Page Toolbar to open the Folders Page.

The Folders Page presents the Folders list for the active Tome.

Tap a Folder name to set the active Folder and open the Bookmarks Page.

Folders are managed via their Action Menu. Tap the Menu button to open the Action Menu. You may reorder, rename, or delete Folders as desired.

To cancel, tap the Back button to return to the Bookmarks Page.

Move/Copy Page

Tap the Move/Copy button from a Bookmarks Page Action Menu to open the Move/Copy Page.

The Move/Copy Page presents the Folders list for the active Tome which do not contain the selected Bookmark.

Move/Copy operations for each Folder are managed via their Action Menu. Tap the Menu button to open the Action Menu. You may move and/or copy the selected Bookmark to the selected Folder.

To cancel, tap the Back button to return to the Bookmarks Page.

Rename Folder Page

Tap the Edit button from a Folders Page Action Menu to open the Rename Folder Page.

Edit the Folder name and tap the Save button to rename the Folder and return to the Folders Page.

To cancel, tap the Back button to return to the Folders Page.

Delete Folder Page

Tap the Delete button from a Folders Page Action Menu to open the Delete Folder Page.

Tap the Delete button to confirm Folder deletion and return to the Folders Page. The active Folder is reset to the next available Folder.

To cancel, tap the Back button to return to the Folders Page.

Search Pane

The Search Pane has three Pages: Results (default), Filters, and History.

Search Expressions

Searches are performed on individual verses. Two search expression types are available: Word or Phrase.

Word search expressions are specified by a single word or a comma separated list of words. Search results will list all verses which contain all search words.

For example, "forgive,sin" will find all verses that contain both "forgive" and "sin".

Phrase search expressions are specified by a word phrase. Search results will list all verses contain that word phrase.

For example, "son of man" will find all verses that contain the phrase "son of man".

Wildcards

Individual words in both Word and Phrase search expressions may contain an optional wildcard character (*) to find variants of a word.

For example, "lord*" will find all occurrences of "lord", "lords", "lord's", "lordly", or "lordship".

Case Sensitivity

To perform a case sensitive search, prepend an at sign (@) to the search expression.

For example, "@LORD*" will find all occurrences of "LORD" and "LORD'S", but not "lord", "lords", "lord's", "lordly", "lordship", "Lord", or "Lord's".

Performing Searches

Tap the Search Pane button to open the Results Page.

Enter a Search Expression in the field at the bottom. Tap the Search Go button or press Enter to set the active Search Expression. The Results Page is opened, presenting results for the active Search Expression.

Results Page

The Results Page Toolbar displays the active Filter and occurrences/verses count.

The Results Page presents a list of verses matching the active Filter in the active Tome. Tap a verse in the list to open the Read Page at that verse.

Filters Page

Tap the Filters button in the Results Page Toolbar to open the Filters Page.

The Filters Page presents a list of occurrences/verses grouped by Tome, Book, or Chapter. Tap on a Tome/Book entry to expand or collapse subentries.

Tap a Filter button to set the active Filter. The Results Page is opened, presenting only matches for the active Filter.

To cancel, tap the Back button to return to the Results Page.

History Page

Tap the History button in the Results Page Toolbar to open the History Page.

The History Page presents a list of previous search expressions. Tap an entry to set the active Search Expression. The Results Page is opened, presenting results for the active Search Expression.

History entries are managed via their Action Menu. Tap the Menu button to open the Action Menu. You may reorder or delete history entries as desired.

To cancel, tap the Back button to return to the Results Page.

Help Pane

The Help Pane has three Pages: Help (default), Settings, and Topics.

Help Page

The Help Page Toolbar displays the name of the active Help Topic.

The Help Page presents help information for the active Help Topic.

Settings Page

Tap the Settings button in Help Page Toolbar to open the Settings Page.

The Settings Page presents a list of user application Settings.

Tap the associated Next/Prev buttons to set the active Font and active Theme.

Tap a Font Size button to set the active Font Size.

Tap the Back button to return to the Help Page.

Topics Page

Tap the Topics button in Help Page Toolbar to open the Topics Page.

The Topics Page presents a list of Help Topics.

Tap a Topic name to set the active Topic and open the Help Page.

Tap the Back button to return to the Help Page.
